Value of $1000 at 5% (or any input) interest rate in 15 years

Use internal JavaScript to implement the following features:

  • User will enter one number for "Interest Rate". Clicking the button will start an animation that changes the value of "Year" and "Money" every second.
  • Every second , "Year" should increase by 1 and "Money" should increase based on the interest rate.
  • After year 2036, "Year" should be reset to 2021 and "Money" should be reset to 1000.
  • Please always keep two decimal places.
